# PDFGoTo Kit

<img width="312" height="312" alt="app_icon" src="https://github.com/user-attachments/assets/384b19dc-136d-4e95-aaa1-e95a8b975e94" />

- **PDFGoTo Kit** is a comprehensive GUI-based PDF toolkit built with Python and Tkinter. It allows you to merge, split, rotate, watermark, stamp, and extract text or images from PDF files easily.
- **Download link** is here: https://drive.google.com/drive/folders/1uEwFcNwE9stRHI7c4k7sd-nqsIKavSXi?usp=drive_link (Version 1.2.0 latest)
- *Please consider donating to the author below if this tool helped you. Your help is greatly appreciated <3*

---

## Features

- **Merge PDFs**: Combine multiple PDF files in a specific order.  
- **Split PDFs**: Split a PDF into individual pages or a custom page range.   
- **Rotate PDFs**: Rotate pages by 90°, 180°, 270°, or custom angles.  
- **Add Watermarks**: Apply text watermarks with customizable opacity.  
- **Add Stamps**: Stamp PDFs with custom text, color, and position.
- **PDF Encryption**: Encrypts your PDF with a password of your own.  
- **Theme Support**: Multiple GUI themes including Blue, Charcoal, Forest, Sunset, and Violet.
- **Converter**: Convert PDF files to DOCX/TEXT/EXCEL format and vice versa. Can convert JPEG/PNG types to PDF format.
- **Metadata**: Read and edit a PDF's metadata.
